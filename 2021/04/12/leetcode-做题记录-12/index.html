剑指 Offer 46. 把数字翻译成字符串

难度中等163

示例 1:

提示：

  • 0 <= num < 231

可以通过递归的方式实现，类似于上台阶问题，一次上一步一次上两步，有多少种走法。

这题也可以分成类似的，一次选一个或者两个去翻译，但是要注意有智能走一步的情况，这就是两个连在一起不在10~25范围内的数字，这时只能走一步。

示例 1:

提示：

  • 0 < grid.length <= 200
  • 0 < grid[0].length <= 200

先初始化dp的第一行第一列。
